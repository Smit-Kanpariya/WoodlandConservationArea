# Woodland Conservation Foundation

> A platform for preserving and showcasing the natural beauty of woodland areas

## Project Overview

Woodland Conservation Foundation is dedicated to preserving and protecting natural woodland areas for future generations. This platform helps raise awareness and support for conservation efforts.

## Getting Started

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   cd Woodland-WCFF
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

### Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
